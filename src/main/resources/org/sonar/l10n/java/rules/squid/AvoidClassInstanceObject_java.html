Avoid instance in Class root block

Restricting declaring/defining variables in class root 

We are restricting the definition/declaration of any variables in class root. 

It is advisable to use local variables while developing web applications.

Compliant Solutions

Follow the steps to fix
 1. Avoid using variables in class root and start using local variable.
 2. If it is really required for declaring variable in class root, reach PM/Lead to add this variable in allowed/exception list .
